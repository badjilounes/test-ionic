#Développement

##Pré-requis
- Création de compte Ionic Pro (version gratuite)
- Initialisation d'une application avec un menu de côté
- Initialisation d'un repertoire git
- Déploiement de l'application sur Ionic View
 
##Sprints
- Login
- Dashboard
- Profile
- Liste medecins
- Détail medecin
- Ajout medecin
- Liste visiteurs
- Détail visiteur
- Liste medicaments
- Prescrire un médicament
